# Memory Matching Game Project

This project aim is to build a modern, high-quality Memory Matching Game using purely HTML, CSS, and JavaScript.

## Stack
- **HTML**: Semantic structure.
- **CSS**: 3D Transforms, Responsive Grid, Modern Aesthetics.
- **JavaScript**: Fisher-Yates shuffle, event-driven game logic.

## Technical Requirements
- 8 cards (4 pairs).
- Icons provided by modern emojis (🍎, 🍌, 🍒, 🍇).
- Flip-back delay: 1000ms.
- Move counter: Incremented per turn (2 clicks).
- Reset: Clears matches, reshuffles cards, resets move count.

## File Breakdown
- `index.html`: Main UI.
- `assets/css/style.css`: Visual and animations.
- `assets/js/script.js`: Interactive gameplay.
